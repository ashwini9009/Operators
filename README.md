# Java Operators Practice

 Project Description
This repository contains basic Java programs created to practice conditional logic and arithmetic operators.

In this project, I solved beginner-level problems using:
- Conditional statements (if-else, ternary operator)
- Arithmetic operators (+, - , *, /, %)
- Logical operators (&&)

Technologies Used
- Java
- Core Java
- JDK

---
Practice Problems Implemented
- Maximum of two numbers
- Minimum of two numbers
- Even or odd number
- Positive or negative number
- Maximum of three numbers
- Leap year check
- Divisibility by 5 and 11
- Check if a number is single digit, double digit, or more
- Check if a number lies between a given range
- Eligibility of voting
- Square and cube
- Swapping numbers using third variable
